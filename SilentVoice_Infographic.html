Sensor Placement

Accuracy depends on physics. We target two specific muscle groups to triangulate phonemes without sound.

Execution Roadmap

A rigorous 4-phase plan designed for the hackathon timeline (Feb 15 - Mar 31).

Feb 15 - Feb 28

Phase 1: Hardware & Signal

- Procurement: Arduino Nano 33 BLE Sense & AD8232.
- Lab: Verify signals on Oscilloscope (clean sine wave).
- Design: 3D print Jaw Headset in Fusion 360.

Mar 01 - Mar 15

Phase 2: Data & Training

- Collection: Record 50 samples/word ("Water", "Pain", "Yes") via Edge Impulse.
- Processing: Apply Spectral Analysis (FFT) & Filters.
- Training: Train 1D-CNN (Convolutional Neural Net) to >85% accuracy.

Mar 16 - Mar 25

Phase 3: Integration

- Deployment: Export TFLite model to C++ library.
- Firmware: Code main Arduino loop (Read -> Inference -> Bluetooth Output).
- App: Build MIT App Inventor dashboard for Text-to-Speech.

Mar 26 - Mar 31

Phase 4: Presentation

- Demo Video: Filming the working prototype.
- Documentation: Finalizing GitHub ReadMe & wiring diagrams.
- Submission: Upload to Hackathon portal.
